微软的2020 Windows 10更新五月版（也称为Windows 10版本2004）已经在5月发布。微软最初面向 “探索者”用户提供此更新，这意味着那些主动检查Windows Update，并手动选择下载该更新的用户可以优先获得更新，当然前提是设备符合兼容性要求。从现在开始，微软 Windows 10版本2004更新可供更多用户和更广泛的硬件配置使用。微软已经悄悄地更新了其发行文档，以确认 Windows 10版本2004的更广泛可用性，但微软表示该更新仅适用于搜索者或在设备上进行手动更新检查的用户。以前，那些在 Windows 10版本1909或1903上手动检查更新的用户无法返回任何信息。现在，如果你有兼容的设备正在运行 Windows 10的6月累积更新补丁，只要没有其他严重问题（比如在系统上发现不兼容的驱动程序），就应该能够下载 Windows 10版本2004更新。微软表示，“ Windows 10版本2004可用于运行 Windows 10版本1903和1909的设备，这些设备通过 Windows Update手动寻求 “检查更新”。我们也正在增加将提供此更新的设备的数量。”用户还应该在运行 Windows 10版本1909或1903的设备上下载并安装 KB4560960补丁。如果在 Windows Update中看不到 KB4560960，请单击 “检查更新”并手动安装补丁。安装此累积更新将使你的设备准备好进行 Windows 10版本2004，但可能无法立即使该更新可用。微软仍在限制 Windows 10版本2004的 Windows Update 交付，如果由于兼容性问题而被 OEM 或芯片制造商阻止，则你将收不到该更新。例如，Windows 10版本2004更新将不提供给具有较旧 Nvidia GPU 驱动程序，以及具有某些 Realtek 蓝牙驱动程序的设备。
