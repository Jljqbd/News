本周三星为2018年 Note 旗舰 Galaxy Note 9推送 One UI 2.1更新，人们预计该更新会首先在韩国推送，但德国用户已经抢先韩国用户收到该更新，在德国推出的是固件版本 N960FXXU5ETF5，包含2020年6月安全补丁。三星 Galaxy Note 9的 One UI 2.1更新并没有带来 Galaxy S20系列上 One UI 2.1的所有功能，但新增了快速分享和音乐分享功能，让用户可以与其他设备分享文件和蓝牙音频连接。在相机应用中，三星增加了单次拍摄模式，当点击快门按钮时，可以拍摄一系列的照片和视频，同时还可以创建自定义滤镜。专业视频录制功能在 One UI 1.0更新时被移除后，现在也已回归。这是一个完整的版本，除了 ISO 和白平衡等参数外，还可以控制快门速度。IT之家了解到，其它市场的 Galaxy Note 9可能还需一段时间才能收到 One UI 2.1更新，而 Galaxy S9和 S9 + 则可能下周开始获得 One UI 2.1。
